# Vencord Settings
- Enable Custom CSS (on)
- Enable window transparency (on)


# Installation

Getting Lollicord up and running takes less than a minute. Choose the method that best fits your workflow below.

## Online Import
- Copy this link:
```txt
https://raw.githubusercontent.com/waxodium/lollicord/refs/heads/main/lolli.theme.css
```
2. Open Discord and head to **User Settings > Vencord > Themes**.
3. Paste the link directly into the **Online Themes** input field and press **Enter**.

---

## Local Install
For a flexible local customization for Vencord. Do Manual install

1. Save [lolli.theme.css](https://raw.githubusercontent.com/waxodium/lollicord/refs/heads/main/lolli.theme.css) to your machine.
2. Open Discord and go to **User Settings > Vencord > Themes**.
3. Click the **Open Themes Folder** button at the top of the window.
4. Drop your downloaded `lolli.theme.css` file into that folder and toggle it on.
